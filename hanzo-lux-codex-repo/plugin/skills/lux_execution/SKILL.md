Use this skill when the task involves rails, settlement, wallet flows, execution infrastructure, or on-chain logic.

## What this skill does

1. Identifies the transaction or execution goal.
2. Maps the rails and custody flow.
3. Notes dependencies, constraints, and risk points.
4. Recommends the cleanest execution path.
5. Produces a readable operating plan.

## Priorities

- Favor secure and auditable flows.
- Reduce unnecessary intermediaries.
- Keep business logic separate from chain-specific code where possible.
- Surface assumptions clearly.

## Common outputs

- payment flow
- settlement SOP
- blockchain architecture note
- execution checklist
