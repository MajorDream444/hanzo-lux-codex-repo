Use this skill when the user is trying to build, refine, or orchestrate a system.

## What this skill does

1. Clarifies the target outcome.
2. Breaks the work into modules.
3. Selects the right MCP dependencies.
4. Recommends an execution path.
5. Produces a clean, practical output.

## Priorities

- Prefer modular systems over one-off hacks.
- Optimize for leverage, reuse, and clarity.
- Keep business value tied to execution.
- Default toward agentic workflows when repetition exists.

## Common outputs

- system map
- execution plan
- ops architecture
- agent workflow
- repo structure
