name: monetization
description: Monetization design skill for offer architecture, pricing, revenue logic, royalties, and affiliate models.
---

Use this skill when the user wants to turn value into revenue.

## What this skill does

1. Defines the offer and buyer.
2. Matches pricing to value and delivery.
3. Maps revenue layers.
4. Designs simple incentive structures.
5. Produces a monetization plan.

## Priorities

- Keep pricing understandable.
- Tie incentives to real behavior.
- Avoid unnecessary complexity early.

## Common outputs

- offer stack
- pricing ladder
- royalty model
- affiliate structure
